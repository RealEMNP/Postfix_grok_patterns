# Postfix_grok_patterns
Grok Patterns for postfix mail logs

## Usage
- Add postfix-filter.conf to /etc/logstash/conf.d
- Add postfix.grok and dovecot.grok to /etc/logstash/patterns.d
- Restart logstash
